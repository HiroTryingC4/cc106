## If You Don't See "GitHub Actions" Option

If you only see "Deploy from a branch":

1. Go to **Actions** tab first
2. Click **"I understand my workflows, go ahead and enable them"**
3. Then go back to **Settings** → **Pages**
4. Now you should see "GitHub Actions" option

## Troubleshooting

### Workflow Not Running?

1. Go to **Actions** tab
2. Click on **"Deploy to GitHub Pages"** workflow
3. Click **"Run workflow"** button
4. Select **"main"** branch
5. Click **"Run workflow"**

### Still Not Working?

The workflow might need permissions:

1. Go to **Settings** → **Actions** → **General**
2. Scroll to **"Workflow permissions"**
3. Select **"Read and write permissions"**
4. Check **"Allow GitHub Actions to create and approve pull requests"**
5. Click **"Save"**
6. Go back to **Actions** and re-run the workflow
